# UI / Структура приложения (вайрфрейм)

Цель: **личный кабинет + окна (разделы)**. Это можно сделать как web-приложение (Telegram WebApp + обычный браузер).

## Общие принципы
- Минимализм (как Notion/Linear): чистый фон, карточки, крупные заголовки.
- Навигация всегда видна: нижнее меню (mobile) или сайдбар (desktop).
- Все действия — 1–2 клика: добавить задачу, создать агента, открыть календарь.

## Навигация (нижняя панель)
1) **Home** — личный кабинет / обзор
2) **Tasks** — задачи
3) **Calendar** — календарь
4) **Agents** — агенты (GPT-шки)
5) **Memory** — память / заметки
6) **Settings** — настройки

> В MVP можно начать с 4 пунктов: Home / Tasks / Calendar / Agents.

---

## Home (Личный кабинет)
**Верх:** приветствие ("Привет, Вован 👋") + статус дня

**Карточки:**
- "Сегодня" — 3 ближайшие задачи/ивента
- "Фокус" — 1 главная задача дня (выбирает пользователь)
- "Агент дня" — быстрый вход в выбранного агента
- "Статистика" — мини: выполнено задач, streak, заметки

**Быстрые кнопки:**
- `+ Задача`
- `+ Событие`
- `+ Агент`

---

## Tasks (Задачи)
**Фильтры:** Сегодня / Неделя / Все / Просрочено

**Список задач:**
- чекбокс
- заголовок
- дата/время
- тег (work, health, content)

**Экран задачи:**
- описание
- подзадачи
- файлы
- связи с календарём
- кнопка "Отдать агенту" (например, "распиши план")

---

## Calendar (Календарь)
**Виды:** День / Неделя / Месяц

**События:**
- создаются голосом или текстом
- можно прикрепить задачу

**Quick add:**
- "Зубной завтра 15:00" → парсинг → событие

---

## Agents (Агенты)
**Список агентов** (карточки):
- имя
- роль
- переключатель "активный" (по умолчанию)
- кнопки: `Чат` / `Настройки`

**Создать агента**:
- имя
- роль (шаблон): Work / Planner / Calories / Content / Music / English
- инструменты (галочки): tasks, calendar, memory, web, files

**Экран агента (чат):**
- чат
- кнопка `🎙 Голос` (записать)
- кнопка `📞 Call` (позже, через отдельный voice pipeline)
- "Что агент знает обо мне" (выжимка памяти)

---

## Memory (Память)
**Виды:**
- Профиль (вкусы, предпочтения)
- Заметки
- Факты

**Кнопки:**
- `+ Заметка`
- `Импорт` (позже)

---

## Telegram бот (UI внутри Telegram)
Бот — входная точка:
- /start
- кнопки: Dashboard, Tasks, Calendar, Agents
- голосовые → транскрипт → агент

---

## Визуальный стиль
- Шрифт: системный (mobile-first)
- Карточки: белые, мягкие тени
- Акцент: один цвет (например, синий)
- Иконки: простые
