# Roadmap (план на 8 недель)

Фокус: **каждый день маленький шаг**. Ты на телефоне — значит всё должно быть максимально простым.

## Неделя 1 — Каркас
- [ ] Репо + структура папок
- [ ] Web: страницы-скелеты (Dashboard/Tasks/Calendar/Agents)
- [ ] API: FastAPI запускается
- [ ] Bot: aiogram запускается
- [ ] 1 сущность: Task (создать/список)

## Неделя 2 — Tasks end-to-end
- [ ] API: CRUD задач
- [ ] Web: добавить/показать задачи
- [ ] Bot: /task add + /tasks list

## Неделя 3 — Calendar
- [ ] API: Event (создать/список)
- [ ] Web: календарь (минимум список + неделя)
- [ ] Bot: команда "добавь событие завтра 15:00"

## Неделя 4 — Agents (конфиги)
- [ ] Агент как конфиг (name/role/tools)
- [ ] Web: создание агента
- [ ] Bot: переключение активного агента

## Неделя 5 — Memory
- [ ] Хранение заметок (structured + raw)
- [ ] Web: Memory экран
- [ ] Bot: "запомни" / "вспомни"

## Неделя 6 — Голос
- [ ] транскрипция голосовых (старт: любой бесплатный вариант/заглушка)
- [ ] агент принимает transcript

## Неделя 7 — Router (главный модератор)
- [ ] intent routing (tasks/calendar/agent chat)
- [ ] логирование действий

## Неделя 8 — Мини-деплой
- [ ] Web на Vercel
- [ ] API на Render/Railway/Fly
- [ ] бот работает через webhook или polling

> После 8 недель: платежи/подписки, умная персонализация, аналитика, конструктор агентов.
